# Анализ рынка заведений общественного питания Москвы

ОПИСАНИЕ ПРОЕКТА

Этот проект посвящен анализу данных рынка заведений общественного питания Москвы. Данные были получены с помощью API сайта открытых данных Москвы. В рамках проекта проводится исследование различных характеристик заведений, таких как их расположение, тип и принадлежность к сетевым компаниям.

ИСТОЧНИК ДАННЫХ:

    Данные взяты из открытых источников — API портала открытых данных Москвы.

ВОЗМОЖНОСТИ ПРОЕКТА:

    Анализ распределения заведений по административным округам и районам

    Исследование типов заведений и их популярности

ПОЛУЧЕННЫЕ ДАННЫЕ:

    global_id (Уникальный идентификатор записи)

    ID(Идентификатор объекта)

    Name(Название заведения)

    IsNetObject(Является ли заведение сетевым)

    OperatingCompany(Оператор заведения)

    TypeObject(Тип заведения (кафе, ресторан и т.д.))

    District(Район)

    Address(Адрес)

    PublicPhone(Контактный телефон)

    SeatsCount(Количество посадочных мест)

    SocialPrivileges(Предоставляются ли льготы)

    Longitude_WGS84(Долгота (координаты WGS84))

    Latitude_WGS84(Широта (координаты WGS84))

    geoData(Географические данные)

ВОЗМОЖНОСТИ ПРОЕКТА

    Оценка доли сетевых заведений

    Анализ доступности льгот в заведениях

    Географическая визуализация данных

ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ

    Python (Pandas, NumPy, Matplotlib, Seaborn, Requests)

    Jupyter Notebook

    API открытых данных Москвы
