# Описание репозитория

Данный репозиторий содержит три проекта, каждый из которых решает отдельную задачу в области анализа данных. 
Все проекты расположены в отдельных папках project1, project2 и project3.

**Cтруктура репозитория**
          
          |-- project1/
          |   |-- pr1.ipynb
          |   |-- Moscow.geojson
          |
          
          |-- project2/
          |   |-- anomaly_detection.ipynb 
          |
          |-- project3/
          |   |-- construction_analysis.ipynb


# Проект 1: Анализ рынка заведений общественного питания Москвы
Описание: Проект направлен на анализ рынка заведений общественного питания в Москве. Включает в себя исследование и визуализацию данных по различным аспектам, таким как расположение, типы заведений и другие характеристики.
Файлы:
**pr1.ipynb** : основной файл с исходным кодом, который выполняет анализ данных и выводит результаты в удобном формате.
Moscow.geojson: файл в формате GeoJSON, который содержит географические данные о районых Москве


# Проект 2: Выявление аномалий в логах просмотров видео
Описание: Проект фокусируется на анализе логов просмотров видео с целью выявления аномальных действий или паттернов, которые могут указывать на проблемы или необычное поведение пользователей. Задача включает в себя предобработку данных, анализ и поиск аномальных событий.
Файлы:
anomaly_detection.ipynb: основной файл с исходным кодом, который содержит алгоритмы для обработки логов и выявления аномалий.

# Проект 3: Анализ фирм, занимающихся строительством жилых и нежилых зданий
Описание: Проект исследует данные о строительных компаниях, которые занимаются проектированием и строительством жилых и нежилых зданий. Анализ включает в себя изучение характеристик компаний, их деятельности,изменение вырочки за 3 кода на основе имеющейся информации распределение по регионам.
Файлы:
construction_analysis.ipynb: основной файл с исходным кодом, который выполняет анализ данных и предоставляет выводы о строительных компаниях, их типах и особенностях деятельности.
