# Анализ логов просмотров видео и выявление аномалий

Описание проекта

Данный проект направлен на анализ датасета логов просмотров видео с целью выявления аномалий, которые не свойственны пользователям. 
Аномалии могут указывать на подозрительную активность, ошибки в системе или необычное поведение пользователей.

Датасет

Датасет содержит следующие поля:

time (время) Временная метка просмотра видео с точностью до миллисекунд.

user_id (ID пользователя) Уникальный идентификатор пользователя, который просмотрел видео.

ip (IP-адрес):IP-адрес устройства, с которого был выполнен просмотр.

video_id (ID видео) Уникальный идентификатор видео, которое было просмотрено.

platform (платформа) Платформа, с которой был выполнен просмотр (например, android, iphone).

Подход к анализу

Очистка данных:

Проверка на дубликаты.

Обработка отсутствующих значений.

Приведение временных меток к единому формату.

Выявление аномалий:

Определение нормального поведения пользователей на основе частоты просмотра одного видео пользователем.

Поиск массовых просмотров видео с одного IP-адреса или устройства.

Обнаружение резкого изменения активности пользователей.

Визуализация данных:

Графики распределения просмотров по времени.

Построение графика boxplot для количества просмотров пользователем одного видео с целью выявления явных выбросов.

Используемые технологии

Python (pandas, numpy, matplotlib, seaborn)

Jupyter Notebook

Выводы

После проведения анализа были выявлены аномальные паттерны поведения, что может быть использовано для улучшения системы рекомендаций,
предотвращения мошенничества и оптимизации работы платформы.
